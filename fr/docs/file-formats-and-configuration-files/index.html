1. RoR File Formats

  • Folder Structure - Organizational pattern of the Rigs of Rods folders

  • Command Line Arguments - very useful for testing vehicles and maps

  • .lua - see LUA System

  • .skin - see Skins

  • .soundscript - see Soundscript files

  • .os - those files are ogre script files. They can contain anything from materials up to caelum sky configurations.

terrain.cfg

http://www.ogre3d.org/wiki/index.php/Terrain_Scene_Manager we have an additional keyword in that file: * disable=1 will disable the terrain mesh and thus hide all the terrain. Collision will be still there.
