1. File Format

File format in general represents a set of attribute-value pairs like:

<attribute>=<value>

For example: Motion blur=Yes

2. Configuration Values

2.1. Graphics

2.1.1. Texture Filtering

Texture filtering that will be used ingame

  • Bilinear

  • Trilinear

  • Anisotropic (best looking)

2.1.2. Sky effects

Type of sky that will be loaded

  • Sandstorm (fastest)

  • Caelum (best looking, slower)

2.1.3. Shadow technique

Shadow technique to be used ingame

  • None

  • Texture shadows

  • Parallel-split Shadow Maps

2.1.4. Water effects

Type of water to use

  • Basic (fastest)

  • Reflection

  • Reflection + refraction (speed optimized)

  • Reflection + refraction (quality optimized)

  • Hydrax

2.1.5. Waves

Adds a lot of fun in boats

  • Yes

  • No

2.1.6. Vegetation

How much grass and how many trees there will be

  • None (fastest)

  • 20%

  • 50%

  • Full (best looking, slower)

2.1.7. Motion blur

Enables the motion blur effect

  • Yes

  • No

2.1.8. Sunburn

Bluish blinding effect

  • Yes

  • No

2.1.9. Mirrors

Enable rear view mirrors in cabin view

  • Yes

  • No

2.2. Controls

2.2.1. Force Feedbac

  • Enable

  • Disable

2.2.2. Deadzone

  • Decimal between 0 and 0.5 (0.1 = 10%, 0.5 = 50%, default: 0.1)

2.3. Advanced

2.3.1. Lights

  • None (fastest)

  • No light sources

  • Only current vehicle, main lights

  • All vehicles, main lights

  • All vehicles, all lights

2.4. Debug

2.4.1. Input Grab

Determines how RoR controls mouse input * All - always grabs the mouse, cannot leave the screen (alt+tab) * Dynamically - grabs the mouse while the window is in focus, can leave screen (alt+tab) * None

2.5. Network

2.5.1. Network enable

  • Yes

  • No

2.5.2. Server port

  • Integer value

2.5.3. Server name

  • String

2.5.4. Nickname

  • String

3. Abstracted by the configurator

  • "Caelum Fog Density", possible Values: 0.05-0.000001 (how dense the fog with Caelum sky is, default: 0.005)

  • "Sandstorm Fog Start", possible Values: 100-10000 (where the fog with Sandstorm starts, default: 500)

4. Sample

A RoR.cfg with lowest settings:

Texture Filtering=None (fastest)
Sky effects=Sandstorm (fastest)
Shadow technique=No shadows (fastest)
Water effects=None
Waves=No
Engine smoke=No
Dust=No
Spray=No
Custom Particles=No
Dashboard=No
Mirrors=No
Sunburn=No
Bloom=No
Motion blur=No
3D Sound renderer=No sound
Threads=1 (Standard CPU)
Enhanced wheels=No
FarClip Distance=100
Fog=No
Lights=None (fastest)
Vegetation=None (fastest)
Screenshot Format=jpg (smaller, default)
Network enable=No
